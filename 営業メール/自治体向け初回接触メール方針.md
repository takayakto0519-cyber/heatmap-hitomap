# 自治体向け初回接触メールの方針（ペルソナC：関係人口の「入口はあるが出口がない」自治体担当者）

作成：2026-07-22。標茶町（北海道）からの返信（丁重なお断り：「既存の体制で運用できており、新規導入を検討する時期ではない」）を教訓に策定。
municipality_profiles.email_draft を今後AIが生成・更新する際は、必ずこの方針に沿うこと。

## このペルソナの正体

関係人口・移住定住の「入口施策」（お試し住宅、ふるさと納税、体験ツアー、SMOUT登録など）は、ほぼ全ての自治体がすでに何かしら持っている。彼らに欠けているのは入口ではなく、**一度きりの来訪を「もう一度関わりたい」に育てる出口**。ヒトマップはここに刺さる。

「新しいシステムの導入」として伝えると、標茶町のように「今は時期じゃない」で弾かれる。**置き換えではなく、今ある取り組みに重ねるだけ**、という一点を必ず伝えること。

## 書き方の4原則

1. **相手の既存施策を先に肯定する**。否定・比較（「〜が足りていません」等）から入らない。まず「拝見しました」「素晴らしい取り組みだと感じました」で始める。
2. **「置き換えではなく重ねるだけ」を明言する一文を必ず入れる**。例：「すでにある〇〇の仕組みを変えていただく必要はありません。そこに、訪れた人の"感じたこと"を可視化する一枚を重ねるだけです。」
3. **CTAは「15分の面談」より軽くする**。第一候補は「まずは資料だけでもご覧いただけないか」。面談を求める場合も「ご都合が合えば」「情報交換程度で」という温度に抑える。予約リンク（https://hitomap.com/schedule）を併記し、返信のハードルを下げる。
4. **署名は「ヒトマップ　加藤貴也」に統一する**。過去のドラフトで「ヒトマップ運営一同」等の揺れがあった（丸亀市版）。文体・結びも含め、この方針書のテンプレートに揃える。
